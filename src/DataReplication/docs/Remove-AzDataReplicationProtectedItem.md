# Remove-AzDataReplicationProtectedItem

## SYNOPSIS
Removes the protected item.

## SYNTAX

```
Remove-AzDataReplicationProtectedItem -Name <String> -ResourceGroupName <String> -VaultName <String>
 [-SubscriptionId <String>] [-ForceDelete] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Removes the protected item.
